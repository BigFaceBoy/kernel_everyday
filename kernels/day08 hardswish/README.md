# HardSwish
$$
标准定义: HardSwish(x)=x \times \frac{RELU6(x+3)}{6}
$$
$$ 分段函数计算: 
HardSwish(x)=\begin{cases}
0 &\text{if x <= -3} \\
\frac{x(x+3)}{6} &\text{if -3 < x <3}\\
x &\text{if x >= 3}
\end{cases}
$$
